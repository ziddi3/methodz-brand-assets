## Repository Rules

- This repository is the official source of all Methodz brand assets.
- Use these assets before creating new logos.
- Do not overwrite approved branding.
- Keep filenames lowercase with hyphens.
- Prefer PNG for transparent graphics.
- Keep folder structure consistent across projects.
